# 🌿 Dayal Food Stuffs - E-Commerce Website

A complete modern e-commerce website for **Dayal Food Stuffs** (Spices & Masala Store), built with **Next.js**, **Tailwind CSS**, **Redux Toolkit**, and Admin Dashboard.

---

# 🚀 Live Features

## 👤 User Side

- Premium Homepage UI
- Hero Banner
- Categories Section
- Best Selling Products
- Latest Products
- Product Details Page
- Add to Cart
- Quantity Update
- Order Summary
- User Login Page
- User Register Page
- My Account Page
- Saved Address Checkout Concept
- Responsive Mobile Design

---

## 🛒 Shopping Features

- Product Search
- Category Filter
- Price Display
- Offer Percentage
- Cart System
- Order Page
- Address Management

## 🔐 Admin Panel

- Admin Login
- Protected Dashboard
- Logout
- Sidebar Navigation
- Revenue Cards
- Orders Chart
- Product Management Ready
- Orders Management Ready

# 🧑‍💻 Tech Stack

- **Next.js 15**
- **React.js**
- **Tailwind CSS**
- **Redux Toolkit**
- **Lucide Icons**
- **LocalStorage Auth (Current Demo)**
- **Future Ready for PostgreSQL + Prisma**
